# COS漫展相册

一个面向 cosplay 漫展照片管理的安卓应用草案，提供基础的选图、修图、9 宫格制作与分享能力。

## 功能亮点

- **快速选图**：通过系统文件选择器读取 SD 卡与内部存储中的图片。
- **轻量修图**：一键添加柔光叠层，适合快速调整氛围。
- **9 宫格生成**：自动将最近选择的 9 张照片拼接为 9 宫格。
- **便捷分享**：直接调用系统分享面板转发生成的图片。
- **相册速览**：横向滑动浏览已选照片，快速切换预览图。
